getAuthentificationData

  • getAuthentificationData(): Promise<getAuthentificationDataResponse | any>
    • Defined in mykoobAPI.ts:82

    Get mykoob access token and other authentication data.

    Returns Promise<getAuthentificationDataResponse | any>

    Returns object with authentification data

    Example

    const api = new MykoobAPI({
    email: 'email@outlook.com',
    password: 'qwerty123'
});

api.getAuthentificationData().then(data => {
    console.info(data.access_token); // '0a1b3c4d......'
    console.info(data.user_id);    // 12345
    console.info(data.expires_in); // 9876543
});
